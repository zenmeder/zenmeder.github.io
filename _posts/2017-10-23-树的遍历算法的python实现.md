#树的遍历算法的python实现
### 一、递归实现前中后序遍历
#### 1. 中序
```python
def inorder(root):
    if not root:
        return []
    if not root.left and not root.right:
        return [root.val]
    res = inorder(root.left)
    res.append(root.val)
    res += inorder(root.right)
    return res
```
#### 2.前序
```python
def preorder(root):
    if not root:
        return []
    if not root.left and not root.right:
        return [root.val]
    res = [root.val]
    res += preorder(root.left)
    res += preorder(root.right)
    return res
```
#### 3.后序
```python
def postorder(root):
    if not root:
        return []
    if not root.left and not root.right:
        return [root.val]
    res = postorder(root.left)
    res += preorder(root.right)
    res.append(root.val)
    return res
```
